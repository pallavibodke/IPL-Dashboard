# IPL-Dashboard
An interactive and visually rich IPL Analysis Dashboard built using Power BI to analyze Indian Premier League match data. This dashboard provides insights into team performance, player statistics, tournament records, toss decisions, batting and bowling achievements, and overall season trends.

-<a href="https://github.com/pallavibodke/IPL-Dashboard/blob/main/deliveries.csv%20(1).zip" >deliveries Dataset</a>
